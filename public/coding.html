Solving the Eikonal Equation with the Fast Marching Method
This was a class project that focused on advanced topics in scientific computing. This project implemented the Fast Marching Method (FMM) to solve the eikonal equation. The eikonal equation shows how fast information travels to an exit set. The FMM uses front propagation over a discretized domain to solve the eikonal equation.

Matlab

Throwback code from an Advance Programming Topics course in 2010. This is code for the classic MineSweeper game that gives visual clues to the user whether or not they have found a bomb or a safe square. Tic Tac Toe
Last piece throwback code from the Advance Programming Topics course in 2010. This is Tac Tac Toe - human v. computer. Users start the game picking whether they think going first or second will give them more of an advantage, picking whether they want to be X's or O's, and finally pick the level they want to play at. The higher the level they pick, the more intelligent the computer AI becomes thus making it harder for the user to win the game.

Java
